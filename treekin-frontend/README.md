Welcome to TreeKin! This is a social media application concept focused on environmentalism, specifically centered around tree plantation and adoption. The goal is to create an attractive and engaging user interface that inspires users to connect with nature and each other.

## ‚ú® Features

The application's user interface is designed to be modern, attractive, and intuitive, drawing inspiration from popular social media apps like Instagram and Snapchat.

*   **Modern Social Media UI:** A profile-centric layout with a clean, tabbed interface for easy navigation.
*   **Vibrant Theme:** A fresh and appealing color scheme using white, green, and yellow to evoke a sense of nature and positivity.
*   **Cinematic Animations:**
    *   Subtle, slow-moving gradient background for a dynamic feel.
    *   Graceful fade-in and slide-up animations for a professional look.
    *   Interactive hover effects on buttons.
    *   Smooth, animated transitions between tabs.
*   **Thematic Design:** A subtle leaf pattern is integrated into the card backgrounds to reinforce the environmental theme.
*   **Core Functionality:**
    *   **User Profiles:** View and manage user profiles.
    *   **User Management:** An admin-like panel to select existing users or create new ones.
    *   **Content Tabs:** Separate sections for "Trees", "Posts", "Profile Editor", and "Credits".

## üöÄ Getting Started

Follow these instructions to get the frontend up and running on your local machine.

### Prerequisites

*   [Node.js](https://nodejs.org/) (v18.x or higher recommended)
*   A package manager like `npm`, `yarn`, or `pnpm`.

### Installation

1.  Clone the repository to your local machine.
2.  Navigate to the frontend directory:
    ```sh
    cd e:\TreeKin\treekin-frontend
    ```
3.  Install the dependencies:
    ```sh
    npm install
    ```

### Running the Application

1.  **Backend Server:** This frontend application requires a backend server to be running. The frontend makes API calls to endpoints like `/api/profiles-page/users`. Please ensure your backend server is running before starting the frontend.

2.  **Frontend Development Server:** Once the backend is running, start the Vite development server:
    ```sh
    npm run dev
    ```

3.  Open your browser and navigate to `http://localhost:5173/`.
